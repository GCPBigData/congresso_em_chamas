      Sobre Congresso em Chamas

      Congresso em Chamas é um dashboard para acompanhamento dos principais indicadores dos deputados federais atualmente com mandato. Criado em 2020 por Gabriel Robaina utilizando Spring Boot no backend e HTML, CSS e JS no frontend com componentes Bootstrap.
